# mac-cleaner-cli: Інструмент для очищення Mac

**mac-cleaner-cli** — це безкоштовний інструмент з відкритим кодом для очищення кешів, журналів та тимчасових файлів на macOS. Він працює з терміналу і не потребує встановлення.

## Основні можливості

1.  **Сканування системи**: Виявляє категорії файлів, що займають зайве місце, такі як:
    *   Кеш користувача
    *   Дані браузера
    *   Тимчасові каталоги
    *   Системні журнали
    *   Залишки папок збірки розробки

2.  **Інтерактивне видалення**: Після сканування з'являється інтерфейс, де можна вибрати, які саме категорії файлів видалити, контролюючи процес.

3.  **Прозорість та безпека**:
    *   Користувач бачить, що саме видаляється.
    *   Процес працює офлайн, не передаючи системну інформацію.
    *   Не торкається критично важливих системних файлів.

## Як використовувати

Для запуску інструменту потрібно виконати в терміналі команду:
```bash
npx mac-cleaner-cli
```

## Переваги

*   **Ефективність**: Може звільнити кілька гігабайт простору, особливо корисно для розробників.
*   **Контроль**: На відміну від графічних аналогів, дає повний контроль над процесом очищення.
*   **Відкритий код**: Код доступний на [GitHub](https://github.com/guhcostan/mac-cleaner-cli), що дозволяє перевірити його роботу.

Існує також версія для Windows: [windows-cleaner-cli](https://github.com/guhcostan/windows-cleaner-cli).
